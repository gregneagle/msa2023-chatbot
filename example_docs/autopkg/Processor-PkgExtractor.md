## Description
Extracts the contents of a bundle-style pkg (possibly on a disk image)
    to pkgroot.

## Input Variables
- **pkg\_path:**
    - **required:** True
    - **description:** Path to a package.
- **extract\_root:**
    - **required:** True
    - **description:** Path to where the new package root will be created.
